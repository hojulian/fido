fido.world.world

World Objects#

class World(ABC, WorldProtocol)

Represents a world.

Currently this is only compatible with Gazebo.

add#

| add(robot: Robot, x=0, y=0, z=0) -> None

Add a robot to the world.

Internally, this is converted into a gazebo_ros spawn_model call.

prepare_robots#

| prepare_robots() -> None

Prepare prepares all the robots in the world.

set_simulation#

| set_simulation(simulation) -> None

Set the parent simulation.

remove#

| remove(robot: Robot) -> None

Remove a robot from the world.

Internally, this is converted into a gazebo_ros delete_model call.

robots#

| robots() -> List[Robot]

Returns a list of robots.

ros#

| ros() -> Ros

Return internal ROS client.

export_files#

| export_files(path, package, rosbridge_port) -> None

Export files to a given file.

Internally, .rosinstall file is exported to the root of the directory. The launch file is exported to $PATH/src/$PACKAGE/launch.
