# Getting Started

## Supported Platform

Fido currently support Linux, MacOS, and Windows via WSL2.

## Requirements

- Python `>=3.8`
- Docker Engine `>=20.10`

## Installation

*Fido is not yet on PyPi, but will be soon!*

Using pip:

```shell
pip install git+https://github.com/hojulian/fido.git
```

Using Poetry:

```shell
poetry add git+https://github.com/hojulian/fido.git
```
